# Análisis de Priorización de Hipótesis para Optimización de Producto Digital

## Descripción del proyecto

Este proyecto analiza diferentes hipótesis de mejora para una tienda en línea utilizando frameworks de priorización como ICE y RICE con el objetivo de determinar cuáles iniciativas tienen mayor potencial de impacto en el negocio.

---

## Objetivo

Evaluar y priorizar hipótesis de mejora para optimizar el desempeño de un producto digital mediante el uso de frameworks de análisis y experimentación.

---

## Problema de negocio

Las empresas digitales generan múltiples ideas para mejorar sus productos, pero implementar todas no es viable. Es necesario priorizar las iniciativas que generen mayor impacto con los recursos disponibles.

---

## Resultados / Aprendizajes

- Se evaluaron diferentes hipótesis de mejora del producto.
- Se aplicaron frameworks de priorización para identificar iniciativas con mayor impacto potencial.
- Se generaron recomendaciones para la planificación de experimentos.
